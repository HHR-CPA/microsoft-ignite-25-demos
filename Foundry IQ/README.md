## Ignite 2025 Foundry IQ Announcements 
* Foundry IQ points agents to one knowledge base to access multiple sources and is integrated in Agent Service where users can keep their agents on tasks with an all-in-one RAG agent.
* Knowledge bases helps keep your agent on task with an all-in-on RAG expert, access remote or index sources, multi-source agentic retrieval engine, and create a base directly in Foundry portal.
* Knowledge sources access remote or indexed with sources selected and agentic RAG engine as well as automatic index creation, transofrmation, and refresh.
* Agentic RAG engine is an AI-powered retrieval for 40% better relevance OOTB, retrieval steeting, retrieval reasoning effort, iterative search, and answer synthesis.
* Entra ID doc level acces with Microsoft Purview.
